## 文件结构

```
├─dist
├─node_modules
├─public
│
├─src
│ ├─assets       // 图片等静态资源
│ ├─fake-api     // 数据获取接口
│ ├─pages        // 路由组件
│ ├─router       // Vue路由配置
│ ├─style        // 样式文件
│ ├─utils        // 封装的工具方法
│ ├─App.vue
│ ├─main.js
```

# 项目启动

```bash
# 安装依赖
npm install

# 服务启动
npm run serve
```
